# skills

Claude Code 用のスキル集。

## handover-docs — 引き継ぎ資料作成スキル

退職・異動・チーム移管のときに、リポジトリを読み込んで引き継ぎ資料（ハンドオーバー資料）を自動で作成するスキル。

```
handover-docs/
├── SKILL.md                    # スキル本体（調査手順・項目・進め方）
├── TEMPLATE.md                 # 引き継ぎ資料の雛形
└── examples/
    └── PROJECT_KNOWLEDGE_BASE.sample.md  # 架空プロジェクトでの記入例
```

### 使い方

Claude Code に「このリポジトリの引き継ぎ資料を作って」と頼むと、`SKILL.md` の手順に沿って

1. コードベースを体系的に調査し（README / 依存 / CI / インフラ / 外部連携など）
2. `TEMPLATE.md` を埋め、
3. コードから分からない情報（連絡先・権限・運用の暗黙知）を質問または `🖊 要記入` として残し、
4. `PROJECT_KNOWLEDGE_BASE.md` を出力

します。記入後のイメージは `examples/PROJECT_KNOWLEDGE_BASE.sample.md` を参照してください。

### 資料の構成

3 つの大セクションに分けて整理します。

- ① 案件概要 … サイト概要 / 機能・ページ / 案件体制（社内・社外）/ インフラ環境（社内 or 得意先）/ 関連資料
- ② 開発環境・セットアップ … リポジトリ構成 / 技術スタック / フロント・バック・インフラ設定 / セットアップ手順 / 環境変数 / ブランチ運用
- ③ 運用・保守 … 環境URL一覧 / デプロイ手順 / 定常作業 / 既知の問題・注意事項 / トラブルシューティング
