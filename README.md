# skills

Claude Code 用のスキル集。

## handover-docs — 引き継ぎ資料作成スキル

退職・異動・チーム移管のときに、リポジトリを読み込んで引き継ぎ資料（ハンドオーバー資料）を自動で作成するスキル。

```
handover-docs/
├── SKILL.md                    # スキル本体（調査手順・項目・進め方）
├── TEMPLATE.md                 # 引き継ぎ資料の雛形
└── examples/
    └── SAMPLE_HANDOVER.md      # 架空プロジェクトでの記入例
```

### 使い方

Claude Code に「このリポジトリの引き継ぎ資料を作って」と頼むと、`SKILL.md` の手順に沿って

1. コードベースを体系的に調査し（README / 依存 / CI / インフラ / TODO / git 履歴など）
2. `TEMPLATE.md` を埋め、
3. コードから分からない情報（連絡先・権限・運用の暗黙知）を質問または `🖊 要記入` として残し、
4. `HANDOVER.md` を出力

します。記入後のイメージは `examples/SAMPLE_HANDOVER.md` を参照してください。

### 資料の構成

3 つの大セクションに分けて整理します。

- **① 案件概要** … プロジェクト概要 / 技術スタック・アーキテクチャ / 関係者・連絡先 / 関連ドキュメント
- **② 開発環境セットアップ** … リポジトリ構成 / ローカル環境構築 / ビルド・テスト・デプロイ / 設定とシークレットの管理場所
- **③ 運用・保守** … 実行環境・インフラ / 外部サービス・連携 / 定期作業・運用 / 既知の課題・技術的負債 / トラブル対応ランブック / 進行中の作業・TODO
